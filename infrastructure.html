Infrastructure

The CastleCrafters Servers / FlandersCraft Network is known to be a big network of (what used to 12), now 6 seperate Minecraft servers all combined in one big network.

These require a certain amount of hardware to be able to run on. To give you an idea of the monthly costs to keep this network running is here a list of per dedicated server that we have running and later on an extra list of all the software it needs to run to sustain itself.

 

Hardware

 

HP Proliant DL160 Gen 9 #1

  • Ram: 32Gb (4 x 8Gb)
  • CPU: 2 x Intel Xeon CPU E5-2603 v3 1.60GHz, 6 core / CPU
  • Storage: 4 x 200Gb SSD
  • OS: Ubuntu 16.04.6 LTS
  • Network: 
    • NIC1: 1Gb/s, 192.168.1.62
    • NIC2: 1Gb/s, 192.168.1.63
